¿Que aprendi?

En el poco tiempo que llevo en estas clases, aprendi lo que eran las etiquetas y que cada una de ellas tiene su estructura semantica. Aprendi que se necesitan el HTML para darle una estructura a una pagina y el CSS para darle forma y estetica a esta estructura. Tambien aprendi a tener un orden semantico tanto en el HTML como en el CSS para que de esta forma se pueda entender de una manera correcta todos los elementos que se colocaron y su funcion en la pagina. Entendi que no era imposible de comprender la programacion como lo solia hacer un tiempo antes, tan solo se necesita materializar los conocimientos que se infundaron y repasarlos en cada oportunidad disponible.
